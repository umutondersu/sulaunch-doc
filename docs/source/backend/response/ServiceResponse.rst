Service Response
================

**Class**: `ServiceResponse<T>`

1. `Data`
    * **Type**: `T?`
    * **Description**: Represents the data contained in the response. It can be nullable.

2. `Success`
    * **Type**: `bool`
    * **Description**: Indicates whether the service operation was successful. It is set to `false` by default.

3. `Message`
